# HR 招聘助手 — 智能简历筛选 & 企微推送

> 🤖 让 AI 帮你看简历，每天自动出报告

## 它能做什么？

HR 每天收到大量 BOSS直聘简历邮件，逐份打开、阅读、对比岗位要求、做出判断——这个过程重复且耗时。

**HR 招聘助手**自动完成这一切：

📬 **自动采集** — 连接企业邮箱，识别并下载 BOSS直聘简历  
🔍 **智能分析** — 对照你的岗位需求，从学历、经验、技能、项目、风险五个维度评分  
📊 **分类报告** — 输出推荐面试 / 待定观察 / 不匹配三档，附面试问题  
📱 **企微推送** — PDF 报告 + 推荐候选人简历原件，直接发到群里，手机就能看  
⏰ **每日自动** — 定时运行，增量处理，不重复不遗漏  

## 适合谁？

- 中小团队 HR，每天处理 10-50 份简历
- 技术招聘场景（AI / 前端 / 全栈 / 产品经理等岗位）
- 使用 BOSS直聘 + 腾讯企业邮箱 + 企业微信的团队

## 报告长什么样？

### 推荐候选人（详细）
- 候选人姓名、匹配岗位、评分
- 3 条核心亮点
- 风险点提示
- 3 道针对性面试问题

### 其他候选人（速览）
- 一行一人，结论 + 一句话原因
- 不浪费你的时间

## 快速开始

1. **安装技能** — 在 Buda 中安装本 Agent
2. **配置邮箱** — 填写 IMAP/SMTP 连接信息
3. **设定岗位需求** — 编辑 `job_requirements.md`，写清楚你要招什么人
4. **配置企微机器人** — 在群里添加机器人，把 Webhook URL 填入配置
5. **设置定时任务** — 选择每天几点自动运行

完成。之后每天自动出报告，你只需要看结果、约面试。

## 技术特点

- 📄 支持图片型 PDF 简历（OCR 识别）
- 🔒 所有数据在私有空间内处理，不外泄
- 🔄 增量处理，不重复分析已看过的简历
- 📱 PDF 报告格式，手机直接打开
- ⚡ 单次可处理 50+ 份简历
