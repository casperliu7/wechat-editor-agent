**`wechat-publish-pipeline`** 是一个专为自媒体、技术博主和独立开发者设计的 **“一条龙全自动 Agent 技能”**。你只需要一句话指令，Agent 就会自动跑完所有耗时的底层工作，把精美的成品送到你的公众号草稿箱！

## 🛠️ 功能清单 (Features)

本流水线会自动执行以下完整步骤（无缝衔接，无需中途人工确认）：

1. **🤖 自主选题 (Topic Selection)**
   - 自动拉取 GitHub Trending, Hacker News 等技术热点。
   - 过滤重复话题，筛选 48 小时内高价值内容。
2. **✍️ 研究与写作 (Research & Write)**
   - 自动抓取外网原网页，绕过反爬机制。
   - 自动精翻核心素材，并采用“第一人称/第三人称”视角撰写原创深度文章，非机械翻译！
3. **🎨 智能文章配图 (Article Illustrations)**
   - 分析文章结构，在“技术架构、数据对比、事件时间线”等关键位置自动插入生成提示词。
   - 调用图像生成引擎批量绘制高质量图解。
4. **🖼️ 封面图生成 (Cover Image)**
   - 自动提炼文章摘要，生成适配微信 16:9 或 2.35:1 的精美头图。
5. **✨ 微信级 HTML 排版 (Markdown → HTML)**
   - 将 Markdown 转为自带样式的 HTML。
   - 自动处理微信特有的“外链转底部脚注”规则，代码高亮、数学公式一应俱全。
6. **🚀 一键投递草稿箱 (Publish to Draft Box)**
   - 调用微信接口或 CDP 自动化浏览器，将完美的图文合集发送到微信公众号草稿箱。

## ⚙️ 环境配置 (Requirements)

为了保证整套工作流顺畅运行，请确保：
1. **安装了配套技能**：本技能需要依赖 `baoyu-post-to-wechat`, `baoyu-markdown-to-html`, `baoyu-article-illustrator` 及相关的 AI 图像大模型技能。
2. **公众号授权配置**：配置好微信公众号的登录缓存 (Cookie/Token) 或开启 Chrome CDP 以接管已登录的浏览器环境。
